图标样式示例

1、引用css

<link rel="stylesheet" type="text/css" href="/assets/res/css/icons.css"/>

2、如何使用

可以使用span或a标记来创建，所有图标的大小均为20px × 20px，如：

<span class="icon_add" title="新增"></span>

使用a标签的好处是当css样式加载失败时，仍然有文字代替

<a href="javascript:void(0);" class="icon_add" title="新增">新增</a>

3、统一图标样式的优势

  • 所有图标集中在一个图片文件上，易于维护和管理
  • 减少http请求
